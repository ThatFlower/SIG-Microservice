# Circuit breaker（熔断降级）

```yaml
kind: Circuitbreaker
metadata:
  name: default
  namespace:
  service:
  interface:
spec:
  type: outbound
  destination:
  - namespace:
    service:
    interface:
  - namespace:
    service:    
    interface:
  match:
  - key: error
    durdsdds:
    throusdd:
  - key: dsdd
    throusdd:
    dsdasdad:
  halfOpen:
    du
  faultDetector: name
```
